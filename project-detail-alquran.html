Al-Quran App

Android
Kotlin
Jetpack Compose
Room Database
MVVM
Dagger Hilt

Overview

A modern Android application for reading and listening to Al-Quran, built with Kotlin and Jetpack Compose. The app provides a beautiful and user-friendly interface for Muslims to access the holy book.

Features

  • Read Al-Quran with Indonesian translation
  • Listen to audio recitations
  • Bookmark favorite verses
  • View reading history
  • Search surah functionality

Technical Details

  • Built with Kotlin and Jetpack Compose for modern UI
  • MVVM architecture pattern
  • Room Database for local storage
  • Dagger Hilt for dependency injection
  • Retrofit for API calls
  • MediaPlayer for audio playback
