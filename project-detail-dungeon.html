Dungeon Crawler

Java
Object-Oriented Programming
Game Development
Console Application

Overview

A text-based dungeon crawler game developed in Java, featuring RPG elements and turn-based combat. Players explore dungeons, battle monsters, and collect treasures in this console-based adventure.

Features

  • Character creation and progression
  • Turn-based combat system
  • Random dungeon generation
  • Inventory management
  • Multiple enemy types
  • Save/Load game functionality

Technical Details

  • Developed in Java
  • Object-oriented design patterns
  • File I/O for game saves
  • Random number generation for encounters
  • Console-based user interface
  • Modular code structure
